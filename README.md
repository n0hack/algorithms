# Is Algorithm Important?

<p align="justify">
옛날에는 <code>'자료구조나 알고리즘을 도대체 왜 공부할까?'</code>라는 생각을 종종 했습니다. 하지만 이 생각은 <a href="https://programmers.co.kr" target="_blank">프로그래머스</a>에서 진행한 위클리 코드 챌린지를 한 번 참여해 보면서 완전히 바뀌게 되었습니다. 당시 문제들을 하나도 풀지 못했으며, 간단해 보이는 문제조차 해결하지 못했습니다. 그렇게 모든 시간을 소모하게 되고 첫 챌린지는 막을 내렸습니다.
<br><br>
얼마 지나지 않아 결과가 나오고, 다른 분들이 어떻게 풀었는지도 공개 되었습니다. 궁금해서 몇 개 찾아 봤는데, 코드들을 본 순간 챌린지에 참여한 모든 분들이 정말 뛰어나 보였고, 코드를 저렇게 짤 수 있다는 것이 부러웠습니다. 그래서 이제는 알고리즘의 중요성을 깨닫고, 열심히 공부하고 있습니다.
</p>

<br>

> 부족한 코드는 언제든 PR을 통해 피드백 주시면 감사하겠습니다.

## Directory

- 문제풀이: ACMICPC(백준), 프로그래머스, 리트코드 문제 풀이
- 알고리즘: 자료구조를 활용한 알고리즘 + 많이 사용되는 알고리즘(정렬, 검색)
- 자료구조: 다양한 자료구조(스택, 큐, 트리, 그래프)
