# Is Algorithm Important?

<p align="justify">
옛날에는 <code>'자료구조나 알고리즘을 도대체 왜 공부할까?'</code>라는 생각을 종종 했습니다. 하지만 이 생각은 <a href="https://programmers.co.kr" target="_blank">프로그래머스</a>에서 진행한 위클리 코드 챌린지를 한 번 참여해 보면서 완전히 바뀌게 되었습니다. 당시 문제들을 하나도 풀지 못했으며, 간단해 보이는 문제조차 해결하지 못했습니다. 그렇게 모든 시간을 소모하게 되고 첫 챌린지는 막을 내렸습니다.
<br><br>
얼마 지나지 않아 결과가 나오고, 다른 분들이 어떻게 풀었는지도 공개 되었습니다. 궁금해서 몇 개 찾아 봤는데, 코드들을 본 순간 챌린지에 참여한 모든 분들이 정말 뛰어나 보였고, 코드를 저렇게 짤 수 있다는 것이 부러웠습니다. 그래서 이제는 알고리즘의 중요성을 깨닫고, 공부하기 시작했습니다. 여전히 어렵고 풀리지 않는 문제들이 많지만, 최대한 이해하고 넘어갈 수 있도록 항상 노력하고 있습니다.
</p>
