# algorithms

<p align="justify">

좋은 코드는 좋은 알고리즘을 바탕으로 만들어진다 생각합니다. 옛날에는 <code>자료구조나 알고리즘과 같은 걸 왜 공부할까?</code>와 같은 생각을 했었습니다. 하지만 이 생각은 얼마 지나지 않아 바꼈는데요. 같은 문제를 풀더라도 다른 사람들은 저보다 훨씬 짧으면서, 가독성까지 좋은 코드를 짠 걸 많이 봤습니다. 저렇게 코드를 예쁘고, 때로는 멋있게 짤 수 있는 능력이 너무 부러웠습니다.
<br><br>
그래서 저도 효율적인 코드를 위해, 알고리즘 공부를 시작하려 합니다.
