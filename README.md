# Is Algorithm Important?

<p align="justify">
옛날에는 <code>'자료구조나 알고리즘은 도대체 왜 공부할까?'</code>라는 생각을 종종 했었어요. 하지만 이 생각은 온라인 문제풀이 사이트인 <a href="https://programmers.co.kr" target="_blank">프로그래머스</a>에서 진행한 위클리 코드 챌린지를 한 번 참여해 보면서 완전히 바뀌게 되었습니다. 모든 시간을 투자했음에도 간단해 보이는 문제조차 해결하지 못했거든요. 그렇게 첫 챌린지가 막을 내렸습니다.
<br><br>
얼마 지나지 않아 결과가 나왔고, 다른 분들이 어떻게 풀었는지도 공개 되었습니다. 궁금해서 몇 개 찾아 봤는데, 코드들을 본 순간 챌린지에 참여한 모든 분들이 정말 뛰어나 보였고, 코드를 아름답게 짤 수 있다는 것이 부러웠습니다. 그래서 이제는 알고리즘의 중요성을 깨닫고, 꾸준히 공부하기 시작했습니다. 여전히 어렵고, 시간을 들여도 잘 풀리지 않는 문제들이 많지만, 최대한 이해하고 넘어갈 수 있도록 항상 노력하고 있습니다.
</p>

> 이슈 또는 PR을 통해 코드 피드백 주시면 감사하겠습니다! 🙂

## Directories

- 문제풀이: 백준, 프로그래머스, 리트코드 등의 Online Judge 문제 풀이
- 알고리즘: 자료구조를 활용하거나 정렬, 탐색처럼 자주 사용되는 알고리즘 정리
- 자료구조: 스택, 큐, 트리, 그래프 등의 자료구조 정리

> 타입 챌린지를 제외한 모든 코드는 자바스크립트로 작성됩니다.
