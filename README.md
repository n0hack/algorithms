## 디렉토리

- alg: 대중적인/기초적인 알고리즘 코드가 저장될 곳
- ds: 기본적인 자료구조 코드가 저장될 곳
- oj: 여러 문제풀이 사이트의 문제를 해결한 코드가 저장될 곳
